# GitHub

**GitHub** ist eine Plattform zum Hosten von Git-Repositories.

**Nutzen für Blogger:**
- Backup deiner Posts
- Versions-Historie
- Zusammenarbeit möglich
- CI/CD für automatisches Deployment

**Alternative:** GitLab, Bitbucket

**Kostenlos:** Für öffentliche und private Repos
